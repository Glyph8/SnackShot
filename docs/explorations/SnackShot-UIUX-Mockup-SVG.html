※ 웹에서 SVG 필터로 텍스처를 흉내 냈습니다. 앱에선 react-native-svg(설치됨)로 동일 기법 + reanimated/expo-haptics 모션·촉각을 입힙니다.

CSS → SVG, 무엇이 달라졌나

"이미지 없이는 여기까지"라는 한계를 SVG 필터가 밀어냅니다. 아래는 전부 래스터 이미지 0장.

🧻 종이 섬유결CSS 도트 → feTurbulence(fractalNoise) 멀티옥타브 노이즈를 multiply로 오버레이. 진짜 종이 결.
🩹 찢긴 모서리매끈한 둥근 사각 → feDisplacementMap으로 카드·테이프 가장자리를 불규칙하게 변위. 손으로 자른 느낌.
✒ 손그림 일러스트이모지/도형 → 잉크 떨림 필터(#ink)를 입힌 SVG path 스케치. 빈 상태가 "노트의 낙서"로.
🖍 거친 형광펜매끈한 직사각형 → #marker 변위로 마커가 번진 가장자리. 손으로 그은 강조.
📌 입체 압정플랫 원 → radialGradient 구체 + 핀대 + 캐스트 섀도. 컨펌 시 박히듯 등장.
♻ 앱으로 직행이 모든 기법이 react-native-svg에서 동일하게 동작 → 목업이 곧 구현 스펙.
